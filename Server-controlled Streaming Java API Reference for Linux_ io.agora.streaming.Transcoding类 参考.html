详细描述


Transcoding 定义直播转码的相关配置。 


类成员变量说明



◆ width






      

        

          int io.agora.streaming.Transcoding.width
        

      




用于服务器控制推流的输出视频的宽度(px)。需要大于 0 px。默认值为 360。 








◆ height






      

        

          int io.agora.streaming.Transcoding.height
        

      




用于服务器控制推流的输出视频的高度(px)。需要大于 0 px。默认值为 640。 








◆ videoGop






      

        

          int io.agora.streaming.Transcoding.videoGop
        

      




输出视频的 GOP(帧)。默认值为 30。 








◆ videoFrameRate






      

        

          int io.agora.streaming.Transcoding.videoFrameRate
        

      




输出视频的帧率。单位为帧每秒，取值范围为 [1, 60]。 服务器会将低于 1 的帧率设置改为 1，将高于 60 的帧率设置改为 60。默认值为 15。 








◆ videoBitrate






      

        

          int io.agora.streaming.Transcoding.videoBitrate
        

      




输出视频的码率(Kbps)。 不能小于 1 Kbps。默认值为 400。 








◆ videoCodecProfile






      

        

          VideoCodecProfileType io.agora.streaming.Transcoding.videoCodecProfile
        

      




输出视频的编解码规格: VideoCodecProfileType。


可以设置为 BASELINE、MAIN 或 HIGH；如果设置其他值，服务端会统一设为默认值 HIGH。 








◆ backgroundColor






      

        

          int io.agora.streaming.Transcoding.backgroundColor
        

      




输出视频的背景色，格式为 RGB 定义下的十六进制整数，不要带 # 号，如 0xFFB6C1 表示浅粉色。默认0x000000，黑色。 








◆ backgroundImageUrl






      

        

          String io.agora.streaming.Transcoding.backgroundImageUrl
        

      




输出视频流的背景图 URL 地址。


注解
如果背景图的 URL 地址无效，5s 后背景色会取代背景图。 








◆ audioSampleRate






      

        

          AudioSampleRateType io.agora.streaming.Transcoding.audioSampleRate
        

      




用于推流直播的输出音频的采样率(Hz): AudioSampleRateType。 








◆ audioBitrate






      

        

          int io.agora.streaming.Transcoding.audioBitrate
        

      




用于推流直播的输出音频的码率(Kbps)。取值范围为[1,128]。默认值为 48。 








◆ audioChannels






      

        

          int io.agora.streaming.Transcoding.audioChannels
        

      




输出音频的声道数： 
    

  • 
1：(默认)单声道 
    • 

  • 
2：双声道 
    • 

  • 
3：三声道 
    • 

  • 
4：四声道 
    • 

  • 
5：五声道 
    • 









◆ lowLatency






      

        

          boolean io.agora.streaming.Transcoding.lowLatency
        

      




低延时模式开关。选填参数。 
    

  • 
true：低延时，不保证画质； 
    • 

  • 
false：(默认)高延时，保证画质。 
    • 









◆ mixAllAudioInChannel






      

        

          boolean io.agora.streaming.Transcoding.mixAllAudioInChannel
        

      




频道内用户的音频混音输出选项。选填参数。


    

  • true：将所有用户的音频混音输出；
    • 

  • false：(默认)根据 TranscodingUser 信息选择用户音频输出。 
    • 









◆ transcodingUsers






      

        

          ArrayList<TranscodingUser> io.agora.streaming.Transcoding.transcodingUsers
        

      




详见 TranscodingUser。 








◆ userConfigExtraInfo






      

        

          String io.agora.streaming.Transcoding.userConfigExtraInfo
        

      




预留参数(选填)：用户自定义的发送到服务器控制推流客户端的信息。用于填充视频中 SEI 帧内容。长度限制 4096 字节。关于 SEI 的详细信息， 详见 SEI 帧相关问题。
